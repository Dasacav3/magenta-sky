## Acerca de Magenta Sky Store

Magenta Sky Store es el prototipo de tienda online creado a partir de la necesidad de poder administrar la venta de productos de la empresa Magenta Sky.

## Tecnologias usadas

- Laravel v8^
- UI v3^
- Tailwind CSS v2^
- SwiperJS v6.8^
